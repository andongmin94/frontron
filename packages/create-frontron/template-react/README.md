# Frontron React Template

This project is generated by `create-frontron` and uses the `frontron` runtime package.

## Stack

- React + Vite + TypeScript
- Electron (`main` + `preload`) with typed bridge
- Tailwind CSS + UI component set

## Scripts

```bash
npm run dev    # renderer only (Vite)
npm run app    # renderer + Electron
npm run build  # renderer build + Electron compile + packaging
npm run lint
```

## Runtime Integration

Electron bootstrap logic lives in `frontron`, not in this template.

- `src/electron/frontron.config.ts`: app runtime configuration
- `src/electron/main.ts`: starts `startFrontronApp`
- `src/electron/preload.ts`: exposes `window.electron` bridge

## IPC Channels

Default channels used by the title bar:

- `window:minimize`
- `window:toggle-maximize`
- `window:hide`
- `window:state`
- `window:maximized-changed`

Legacy channels are also supported for migration.
