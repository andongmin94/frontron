# Frontron Next Template

This project is generated by `create-frontron` and uses the `frontron` runtime package.

## Stack

- Next.js App Router + TypeScript
- Electron (`main` + `preload`) with typed bridge
- Tailwind CSS + UI component set

## Scripts

```bash
npm run dev     # Next dev server
npm run app     # Next dev + Electron
npm run build   # Next build + Electron compile + package
npm run lint
```

## Runtime Integration

Shared Electron runtime logic lives in `frontron`.

- `electron/frontron.config.ts`: app runtime configuration
- `electron/main.ts`: starts `startFrontronApp`
- `electron/preload.mts`: exposes `window.electron` bridge

## IPC Channels

Default channels used by the title bar:

- `window:minimize`
- `window:toggle-maximize`
- `window:hide`
- `window:state`
- `window:maximized-changed`

Legacy channels are also supported for migration compatibility.
